Problem
No Immutable Backups
Client had traditional tape + NAS backup with no immutability. A simulated ransomware attack encrypted all backup repositories within 4 hours. Zero viable recovery points.
Investigation
Attack Surface Analysis
Backup server accessible via SMB share with domain admin credentials. No network segmentation. Veeam service account had excessive privileges across all datastores.
Root Cause
Flat Network + Over-Privileged Backup
Ransomware traversed flat network via SMB. Backup server compromised first — attacker deleted all snapshots before encrypting production. Classic double-encryption pattern.
Remediation
3-2-1-1-0 with WORM
Deployed hardened Veeam Linux repo with XFS immutability. Azure Blob WORM as cloud tier. Least-privilege service accounts. VLAN isolation for backup network. SureBackup automation.

Problem
Fragmented MPLS + VPN
Client operating 4 separate MPLS circuits + legacy SSL VPN. 340ms average latency to cloud resources. MPLS costs ₹12L/month. VPN gateway single point of failure causing weekly outages.

Traffic Flow Analysis
80% of traffic destined for AWS. MPLS routing all cloud traffic via HQ datacenter — hairpinning adding 180ms latency. SSL VPN gateway saturated at peak hours. BGP routing inefficient.
Root Cause
Hub-and-Spoke Hairpin
Legacy hub-spoke design forced branch offices to backhaul all internet and AWS traffic via HQ. No direct cloud connectivity. MPLS costs disproportionate to actual WAN utilisation.
Remediation
AWS Transit Gateway + SD-WAN
Deployed AWS Transit Gateway as network hub. Site-to-Site VPN from all branches. Direct Connect for HQ. Replaced SSL VPN with AWS Client VPN + MFA. BGP route optimisation.

Security Risk Reality

What Happens If You Do Nothing?

Every unaddressed security gap is a chain reaction waiting to happen. This is the realistic attack progression for an organisation without proper security controls.

No MFA Enforced
Passwords alone protect all admin and user accounts. Phishing and credential stuffing succeed silently.
Starting Point
Credential Theft via Phishing
Attacker sends convincing M365 phishing email. User enters credentials. Session token harvested. Attacker now inside your tenant — undetected.
T+0hr
Account Compromise & Privilege Escalation
Attacker enumerates Azure AD, finds Global Admin eligible accounts in PIM with no approval required. Elevates in 4 minutes. Full tenant access achieved.
T+4hr
Backup Infrastructure Targeted
With domain admin, attacker accesses Veeam via SMB share. Deletes all backup restore points. Disables backup jobs. Verifies no viable recovery path exists before detonating ransomware.
T+8hr
Data Exfiltration
Before encryption, attacker exfiltrates customer PII, financial data, and IP via cloud sync. 2TB transferred to attacker-controlled storage. This data will be leaked or sold regardless of ransom payment.
T+12hr
Ransomware Detonation
Ransomware deployed simultaneously to all endpoints and servers via GPO. Encryption completes in 4 hours. No backups. No visibility. Operations completely halted. Ransom demand arrives.
T+16hr
Financial & Operational Collapse
30+ days of downtime. Emergency IR firm engaged at ₹25L+. Rebuild from scratch: ₹40L+. Regulatory fines (DPDP/GDPR): ₹20L–2Cr. Reputational damage: immeasurable. Many SMEs do not survive.
