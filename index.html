AWS Site-to-Site VPN Hybrid Architecture

CASE-003 // STORAGE INFRASTRUCTURE

S3-Compatible Private Cloud Object Storage

Architected a scalable private cloud object storage platform delivering AWS S3-compatible APIs on-premises, replacing legacy NAS with a cost-optimised, scalable solution integrated with existing backup pipelines and compliance requirements.

S3 API MinIO / Ceph Object Storage WORM Compliant
